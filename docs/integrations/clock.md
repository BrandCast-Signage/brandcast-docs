# Clock

Display the current time for any timezone.

**Type:** Native Widget (No external account required)

## Overview
The Clock widget displays the current time. BrandCast automatically creates a default clock for your store's timezone, but you can create additional clocks for other timezones (e.g., "London Office", "Tokyo HQ").

## Configuration
*   **Timezone:** Select any global timezone.
*   **Format:** 12-hour (AM/PM) or 24-hour.
*   **Style:** Digital or Analog (if supported).
*   **Date:** Show/hide date.

## Usage
1.  **Create Source:** Go to **Integrations** > **Clock** > **Create New Source**.
2.  **Add to Layout:** In Layout Editor, add a **Clock** area and select your source.
