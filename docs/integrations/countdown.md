# Countdown

Display a countdown timer to a specific date and time.

**Type:** Native Widget (No external account required)

## Overview
Create excitement for events, sales, or deadlines.

## Configuration
*   **Target Date:** Date and time to count down to.
*   **Timezone:** Timezone of the event.
*   **Format:** Days/Hours/Minutes/Seconds.
*   **Completion Message:** What to show when the timer hits zero (e.g., "Event Started!").

## Usage
1.  **Create Source:** Go to **Integrations** > **Countdown** > **Create New Source**.
2.  **Add to Layout:** In Layout Editor, add a **Countdown** area and select your source.
