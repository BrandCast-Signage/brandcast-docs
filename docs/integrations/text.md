# Text

Display static text, announcements, or simple messages.

**Type:** Native Widget (No external account required)

## Overview
A simple way to add headers, instructions, or welcome messages to your display without needing an external document.

## Configuration
*   **Content:** The text to display.
*   **Typography:** Font family, size, weight, color.
*   **Alignment:** Left, Center, Right.

## Usage
1.  **Create Source:** Go to **Integrations** > **Text** > **Create New Source**.
2.  **Add to Layout:** In Layout Editor, add a **Text** area and select your source.
